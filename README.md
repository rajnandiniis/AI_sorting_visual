
# **🌟 AI Sorting Visualizer 🌟**

## **✨ Overview**  
The **AI Sorting Visualizer** is an innovative tool designed to make sorting algorithms easy to understand. Through interactive visuals and detailed explanations, it helps users grasp the concepts behind popular sorting techniques.

---

## **🚀 Features**  
- **🎨 Real-Time Visualizations**: Step-by-step representation of sorting processes.  
- **📖 In-Depth Explanations**: Includes logic, comparisons, swaps, and metrics for each algorithm.  
- **⚡ Algorithms Covered**:  
  - **Bubble Sort**  
  - **Quick Sort**  
  - **Merge Sort**  
  - **Insertion Sort**  
  - **Selection Sort**  
- **📊 Performance Analytics**: Time complexity and operations tracking.  
- **🌈 Easy-to-Use Interface**: Perfect for beginners and enthusiasts!

## **📚 Understanding Sorting Algorithms**  
### **Bubble Sort**  
- **Best Case**: O(n)  
- **Worst Case**: O(n²)  
- **Description**: Compares adjacent elements and swaps them if needed.  

### **Quick Sort**  
- **Best Case**: O(n log n)  
- **Worst Case**: O(n²)  
- **Description**: Uses a pivot to partition the array and sort subarrays recursively.  

### **Merge Sort**  
- **Best Case**: O(n log n)  
- **Worst Case**: O(n log n)  
- **Description**: Divides the array into halves, sorts them, and merges them back together.
